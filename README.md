## How To Run

<details>
  <summary>TERMUX/UBUNTU/SSH USER</summary>
	
  ```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
git clone https://github.com/dhaffnavyz/DaisuXe-Md
cd DaisuXe-Md
bash install.sh
pkg install yarn
yarn
```
</details>
